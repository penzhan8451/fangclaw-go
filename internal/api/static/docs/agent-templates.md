# Agent 模板使用指南

## 概述

FangClaw 提供了多个预配置的 Agent 模板，可以快速创建适合不同场景的智能代理。

## 可用模板

### 1. Researcher（研究员）
- **类别**: content
- **功能**: 深度自主研究员，交叉验证多源信息，生成带引用的报告
- **适用场景**: 市场调研、技术分析、学术研究

### 2. Lead（潜在客户开发）
- **类别**: productivity
- **功能**: 每日运行，发现符合你 ICP 的潜在客户，富集研究并评分
- **适用场景**: 销售线索挖掘、客户开发

### 3. Collector（情报收集）
- **类别**: data
- **功能**: OSINT 级情报，持续监控目标，检测变化并构建知识图谱
- **适用场景**: 竞品监控、舆情分析

### 4. Predictor（预测分析）
- **类别**: data
- **功能**: 超级预测引擎，收集信号，构建校准推理链，做出预测
- **适用场景**: 趋势预测、决策辅助

### 5. Clip（视频处理）
- **类别**: content
- **功能**: YouTube 视频处理，下载、识别最佳时刻、剪辑成竖版短视频
- **适用场景**: 内容创作、社交媒体

### 6. Twitter（社交媒体管理）
- **类别**: communication
- **功能**: 自主 Twitter/X 账号管理，创作内容、定时发布、回复提及
- **适用场景**: 品牌运营、社媒管理

### 7. Browser（浏览器自动化）
- **类别**: productivity
- **功能**: Web 自动化代理，导航网站、填写表单、处理多步工作流
- **适用场景**: 数据采集、流程自动化

## 如何使用模板

### 方式一：通过 Dashboard 使用

1. 登录 Dashboard
2. 进入 Agents 页面
3. 点击 "New Agent"
4. 选择 "Use Template"
5. 选择合适的模板
6. 自定义配置（可选）
7. 创建 Agent

### 方式二：通过 API 使用

```bash
# 使用模板创建 Agent
POST /api/agent-templates/{template-id}/spawn
```

## 自定义模板

你可以基于现有模板创建自定义 Agent：

1. 选择一个基础模板
2. 修改系统提示词
3. 配置技能和工具
4. 调整模型参数
5. 保存为新 Agent

## 最佳实践

1. **选择合适的模板**: 根据你的具体需求选择最匹配的模板
2. **迭代优化**: 创建后根据实际效果调整配置
3. **组合使用**: 多个 Agent 可以通过工作流组合使用
4. **监控效果**: 定期查看 Agent 的使用统计和效果
