### ✨ 简单示例涵盖的功能
- Sequential : 基础顺序执行
- Conditional : 基于输出内容的条件执行
- Fan-out : 并行执行多个步骤
- Error Handling : fail、skip、retry 三种错误处理模式
- Variable Passing : 使用 output_var 在步骤间传递变量

### 🏢 复杂实际工作场景
1. Software Release Pipeline : 代码审查 → 安全扫描 → 性能分析 → 文档 → 发布说明
2. Content Creation Pipeline : 研究 → 大纲 → 草稿 → 审查 → 修订 → 定稿
3. Market Research Report : 竞品分析 → 市场趋势 → 客户洞察 → SWOT → 建议 → 报告
4. Technical Documentation : 代码分析 → API 文档 → 入门指南 → 教程 → 故障排除
5. Data Analysis Pipeline : 数据评估 → 清洗 → 探索 → 统计 → 可视化 → 洞察 → 建议
6. Customer Support Ticket Handler : 分类 → 分析 → 紧急判断 → 响应草稿 → 审查 → 修订 → 终稿
